Mega Button Tycoon is a Roblox tycoon game where players press a button, earn coins, and upgrade their button over time.

## 🎮 Roblox Game Page

The game is public on Roblox.

Roblox link: https://www.roblox.com/games/110385235020662/Mega-Button-Tycoon

## 🧠 About the Game

Mega Button Tycoon is designed to be simple, colorful, and solo-friendly.

The main gameplay loop is pressing a button, earning coins, and using those coins to unlock upgrades. Players can claim a base, press their button, earn coins, and progress by upgrading their tycoon.

## 🚀 Features

- Button-based coin system
- Coin counter UI
- Upgrade system
- Player base claiming
- Tycoon-style gameplay
- Solo-friendly design
- Colorful Roblox map

## 🚧 Development Status

Mega Button Tycoon is public on Roblox again.
